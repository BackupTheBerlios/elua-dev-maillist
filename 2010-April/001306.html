You definitely don't need to buy any tools. You have two choices here:

1. build your own toolchain (http://www.eluaproject.net/en_tc_cortex.html)
2. use an existing toolchain (for example CodeSourcery).

I'd go with 2 above, it's a much easier option for a beginner (although it
generally leads to slightly larger images). Once you installed the lite
version of the CodeSourcery toolchain (
http://www.codesourcery.com/sgpp/lite/arm/portal/subscription?@template=lite,
you need the EABI version), all you need to do is this:

$ scons cpu=lm3s8962 toolchain=codesourcery prog

Then enjoy :) More details about building are here :
http://www.eluaproject.net/en_building.html#configuring

Best,
Bogdan
