> I found gcinfo, which shows how much is in use.
> Is that out of the whole memory pool?
> e.g., if it shows 5 kb is that out of the total RAM on the
> microcontroller, or does it ignore some of the memory used
> by eLua internally.

'gcinfo' is only Lua object, eLua uses more memory than that for other
stuff. There is currently no way to know how much memory is actually in
use, we might add that in a future version (although this is not a simple
problem because of the way some allocators request memory).
