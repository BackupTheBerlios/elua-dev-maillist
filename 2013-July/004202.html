> Hi! My router is pre-loaded with lua functions. I'm trying to read from
> console what I type and put it into a file in my router. The lua only have
> basic input/output functions like file:read,file:write,io:open,etc. It does
> not have rs232 or urat library.
>
> I wrote the code below:
>        inp = assert(io.open("/dev/ttyS1","r"))
>        out = assert(io.open("rtest.txt","w"))
>     while true do
>        data = inp:read("*all")
>        data = string.gsub(data, "\r\n", "\n")
>        out:write(data)
>      end
>      assert(out:close())
>
> The code does not work. Can anybody help and advise?

Hi
  Well, you should check the values of inp and out to see if they are
nil, and also the value of "data", before and after the "gsub", but
"It does not work" is too vague to know what to suggest.
  Can you say what happens exactly, and how this is different from
what you expected to happen? A transcript of what happens on your
screen might be useful.

For a more complete guide on how to ask technical questions in mailing
lists with precision, and how to get the right answer quickly, there
is a useful article here:
http://www.catb.org/esr/faqs/smart-questions.html  Its tone is rather
rude, but it does say some useful things that are not obvious. It's
also been translated into many languages.
