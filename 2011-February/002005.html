Hello List,
   The eLua project will change its source control versioning system from
Subversion to GIT
(http://git-scm.com/).
   The master repository host will also move from BerliOS to github at
https://github.com/elua/elua.
   This change will become effective on March 1st 2011, three weeks from
now. After this date the eLua
subversion repository will no longer be accessible.

    If you are unfamiliar with git, here is a couple of links to get you
started:

   - Git tutorial for subversion users:  http://git.or.cz/course/svn.html.
   - Git's own help and book: http://help.github.com/ and
   http://progit.org/book/
   - The Git Community Book  http://book.git-scm.com/index.html

Best
