Hello,
   There is a new table in our downloads->binaries section of the doc/site
at http://www.eluaproject.net/en_downloads.html#binaries
   Right below the official latest release binary images downloads table, we
now have a new table with custom images, also ready to be flashed and used
on eLua supported boards.
   Starting with an *LM3S eLua Games edition*, that runs on Luminary Micro
LM3S Cortex-M3 boards *that have an OLED RIT display, *this build uses our
autorun feature to automatically (upon reset or power on) run a simple menu
UI on the OLED display, where you can select among 6~7 eLua examples and
games to play on the board.
   No terminal is required to start the programs. Just reset the board to go
back to the menu, navigate it with the arrow buttons and select an option
with the select button.
   Just download the image from our site on the link above, flash it to you
board using Luminary Micro's flash utility following our tutorial @
http://www.eluaproject.net/en_installing_lm3s.html, reset the board and do
the rest directly on the kit's display and buttons.

   For developers, the LM3S eLuaGames build is on a repo branch @
lm3seluagames.
   As I only have an EK-LM3S8962 board, this is the only supported platform
for now but an EK-LM3S6965 version will probably come soon, as some eLuers
have them.

   The current version includes the following games and examples:

- *Pong* (an eLua incarnation of the classical game)
- *Tetrives* (Tetris in eLua)
- *Snake*
- *Spaceship* (an eLua version of space invaders, still incomplete [no
score] but playable)
- *led* (the blinking led example included in the regular distros. The
embedded "hello world" app :)
- *pwmled* (same as above)
- *logo* (an example of floating point real time calculations for rotating
the eLua logo on the display)

  We hope you enjoy playing eLua games you your LM3S kits!
  Reports are welcome.

Best
Dado
