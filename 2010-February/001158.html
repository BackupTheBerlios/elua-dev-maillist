Hello,
   On platform_conf.h for lpc24xx platform, we have:

// Pins per port configuration:
// #define PIO_PINS_PER_PORT (n) if each port has the same number of pins,
or
// #define PIO_PIN_ARRAY { n1, n2, ... } to define pins per port in an array

// Use #define PIO_PINS_PER_PORT 0 if this isn't needed
#define PIO_PINS_PER_PORT     31

Is PIO_PINS_PER_PORT really what it means or it is actually
pio_pins_per_port - 1 ?

sight.
