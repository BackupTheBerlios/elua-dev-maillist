Yes, the service_id change seems to be correct. Please try the sequence of
steps below:

1. start your eLua board
2. start your terminal emulator on COM21
3. start the RFS multiplexer as above
4. reset your eLua board

If you're getting any input on the terminal emulator after step 4 above, try
to input something and see if that works. If not, please let me know. The
mux server was never tested on XP, only on Win7. Although I kinda doubt
that's the problem. Are you by any chance on 64bit XP?
