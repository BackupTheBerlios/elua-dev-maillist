Hi,

As previously announced, the eLua SVN repository is now in read-only
mode. To access the new git repository simply go to this address:

https://github.com/elua/elua

Choose one of the git URLs (using the "HTTP" or "Git read-only"
buttons) and clone your copy of the repository, then enjoy. That's it
:) The recommended method is "Git read-only" unless you are behind a
problematic firewall, in which case you can try "HTTP" instead.
For developers: if you need write access (git push) please create an
account on github.com and then let us know, we'll make the necessary
arrangements.
Please let us know if you have any problems with this.
