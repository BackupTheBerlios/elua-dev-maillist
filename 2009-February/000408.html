I don't want to start any "wars" here, but I'm curious to what everyone uses
or is interested in using for PCB design.

for Solidworks).  So far I like Eagle.

Blackfine-One, among other projects).

My feeling is it'd be good if everyone could use one or two systems (e.g.
Eagle or gEDA).
