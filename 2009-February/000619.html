Hello,
  A second version of a RAW image format to a RIT OLED display format
converter, can be found at:

http://www.led.giga.puc-rio.br/v2-raw2rit

  This one is a bit more "Lua" :)

For comparison purposes, the old (first version) one is at
http://www.led.giga.puc-rio.br/v1-raw2rit

The input format is .RAW, no header infos, one byte per pixel, 15 gray
levels per pixel and max size 128x96.
The output format is what disp.draw expects at the "img" argument. Actual
size must be specified on width and height params too.
Both solutions use/require() the module bit, for bitwise manipulation, by
Reuben Thomas.
