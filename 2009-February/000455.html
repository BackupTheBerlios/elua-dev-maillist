I added FatFs support to my local version of eLua to run scripts from an
MMC/SD card. The changes use the "/mmc" prefix for the card and you can
place an autorun.lua on the card to load a script upon startup. "/rom" can
still be used for the files in flash. I will post these changes to a
separate trunk in the SVN later today or tomorrow.
