When this port was originally done, I didn't happen to have an ethernet
connector to implement and test with.  I also wasn't able to adapt eLua to
compile in their online compiler since it depends on some C99 features that
weren't available in their setup at the time.  So, the port is a low-level
one not using the mbed libraries.

I don't know that we have an example of an ethernet implementation for
another LPC chip, but it may not be too difficult to add to the port.  If I
recall mbed might have also open sourced some of their libraries under an
MIT licence which could serve as a starting point for this?  Or NXP may
have an example implementation with their peripheral libraries?
