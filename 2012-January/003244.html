I'm not sure if anyone has replied off-list about this.  There are
existing versions of the logo used on the previous revision of the
site that I put together which are available as bitmaps here:
http://wiki.eluaproject.net/Logos

I have a couple different versions of that file, which I originally
edited in Inkscape, then did some work in Illustrator.  The Inkscape
file I have doesn't match the rendered versions so I won't just
include that.  Instead I've attached the illustrator version (.ai),
which is actually in PDF format, and an svg export from Illustrator
which should open in Inkscape (loaded for me just now and looked OK).
One potential caveat is that I used Helvetica as the font, as in the
Lua logo, which you may not have on your system so it might substitute
and render a little differently.

I've also attached a reasonably large png with transparency
(1024x1024) in case that might be useful.

It would be great if we could get a vector version of the latest logo,
including maybe a "flat" version of it (sans the embossing/specular
highlight/shadow) that would easily work in a vector format, but I'm
not sure that's available.  I can certainly adapt the older version to
more closely match the adjusted dimensions on the current one.
