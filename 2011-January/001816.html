Hi!
  We're writing a book for a tiny computer that runs eLua natively and
want to sell it as a free gift included in a book which explains how
to use it. This provides us with the opportunity to write a book that
teaches anything we care to write about as an example of how to use
Lua/eLua with board-specific examples included. As the range of
Mizar32 add-on boards includes a naked set of headers with an are of
interestingly-patterned solder areas, I propose as a first example an
array of LEDs and a program to flash them in different patterns in
response to button presses.

  Contributors are invited to check code.google.com/p/mizar32 and
elua-project.net

   Recommendations for a collaborative writing tool to use? It would
be nice to produce (good-quality typesetting) paper copy too to flog
on lulu.com or wherever.
