Hello eLuers,
   We're proud to announce that we published a new version (v0.2.2) of the
eLua Web Builder.
   The main additions and enhancements are:

- Files upload support now based on PLupload. This was done after some users
complained about dependency on Flash and some issues with different
browsers. eLua Web Builder now identifies what the clients browsers have and
automatically uses html5, Flash, Silverlight and other fancy ways to upload
files. This is completely transparent to users and does not require any
special instalation of plugins or add-ons on the browsers.

- Files Adition, management and upload now more "centralized" on the build
interface. It just makes it easier (less clicks) and, hopefully, intuitive
to handle files for the ROMFS.

- Competed-build message and option to download the image immediately
presented to the user. Built images can still be stored in the user's area
and will stay there until manually deleted by the user but she doesn't need
to go there to download the last build generated anymore.

- Different format options to download the build results. Users can download
only the raw binary image that will be flashed to their kits or, as before,
a zip file that contains this same image together with the build results,
romfs file contents, error/warning log reports etc ....

- A "Contact us"

- Some other minor goodies up to you to discover :)
