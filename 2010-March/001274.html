If you want to have a pretty good idea about eLua using, I suggest you (as a
newbie programmer too, not a 32-bit-monster like these guys) to have a look
at the examples that are in the romfs in a default build.
You can also have a look at the eLua Examples
repo<http://code.google.com/p/eluaexamples/>.
There you may see some examples of games, simple application and some
interesting libs that eLua users have been developing.
There is only code in the repo. No good doc yet, because it's being
organized to be all together in a eLua User's area.

Most part of the code was written by programmers that don't have much
experience in Lua or even in other programming languages. So, it may be
useful for you at this moment.
