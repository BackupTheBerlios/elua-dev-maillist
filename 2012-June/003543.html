There is no need for this at all.  The 16-bit CPU is going to be squeezed, rather like 8-bit CPUs. Sure, 8-bit 8051 is still thriving, but even high-performance vendors such as Silicon Labs have migrated to 32-bit Cortex-M3 devices for their latest set of products...
