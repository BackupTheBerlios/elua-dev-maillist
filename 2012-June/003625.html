So, I've added strftime formatting to the get function.
I had to calc Day of Week and Day of Year in the get function ( probably
not the best place for that ) in order to fill the C time struct ( or else
strftime hangs ).
The get() with no parameter returns strftime("%c") like lua formated as:
Wed Jun 27 15:25:11 2012

Now the get("*t") returns the table and get("something_else") returns a
formatted string.

I've only added this for mbed. I'll update str9 when we decide an interface.
