James Snyder has been one of the greatest contributors to the project since
the start of it. We actually consider him as almost an Author now and this
should/will be reflected on our doc too.
A lot of work has been done this year but not always in a well coordinated
way. This led to James, Bogdan, SimpleMachines' guys, just to mention a
few, to end up working on repos of their choices.
We should go back ASAP to the Pull Request model that git suggests and
merge all these efforts to master.
