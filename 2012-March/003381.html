> "The Go @ IDE" (https://github.com/nuccioraciti/Goat-IDE) is an editor 
> dedicated to the embedded eLua World:
> 
> some features:
> 
> *) Its editor (based on the wonderful library "scintilla" 
> http://www.scintilla.org/), is lightweight and powerful.
> 
> *) Has additional features such as "syntax highlighting", "code folding" 
> and "code completion" based on the eLua API.
> 
> *) Includes the capability to connect with the real hardware (through 
> the serial or the USB port) obtaining the prompt of Elua inside it.
> 
> *) It's able to send the .lua/.lc file on the Hardware (using the eLua 
> xmodem protocol).
> 
> *) Supports some terminal commands (standard ansi/vt100), so it's able 
> to play games like "hanoi" or "hungman" (using the eLua TERM capability).
> 
> *) Includes the for the DFU bootloader programmer (only for the avr32 
> platform for now).
> 
>
