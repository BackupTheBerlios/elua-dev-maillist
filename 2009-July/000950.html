Hi just a note to say it works on this configuration,

   Debian Linux  custom kernel 2.6.29.6  with patch to ftdi_sio driver so it
use's the on-board serial port on the comstick board, and lives the Jtag
port free for openocd.
   build with gcc4.4.0 arm cross compiler I build my self.

   Openocd  is SVN:2551M  configured for FTDI libftd2xx,  version 4.16
   elua is SVN r365    with miner fix for cpu configuration problem
   binutils 2.19.51
   Gcc G4.4.0
   Newlib 1.17.0
   nsight debuger 6.8.50.20090610-cvs
   gdb 6.8


   Had a few problems along the way,   so if any one is trying to get the
same board working
   I can give.. scripts I used to build gcc,  advice..  etc
